Gridbox of Artists 4x4

Introduction

This website created the grid below using CSS display grid. The grid-template-columns were on repeat four times with a size using minmax ranging from auto to one fraction. The grid-template-rows on the other hand were also on repeat four times with a size of three hundred thirty three pixels. We're going to make this introduction distinct by adding a cool border with white glow.
